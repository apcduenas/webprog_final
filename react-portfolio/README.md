# React Portfolio - Ghost Rider Theme

This is a personal portfolio website converted to React + Vite, featuring a "Ghost Rider" theme with a custom fire cursor, video transitions, and scorched parchment design.

## Features
- **Dark Mode (Ghost Rider)**: Fiery red/orange theme with Ghost Rider background.
- **Light Mode**: Blue theme with clean visuals.
- **Fire Cursor**: Interactive flame effect following the mouse.
- **Burned Paper Design**: Sections styled like scorched parchment with organic edges.
- **Interactive Elements**: Gallery lightbox, skills filter, guestbook form.

## How to Run

1.  **Open Terminal**: Navigate to the project folder:
    ```bash
    cd react-portfolio
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **Open in Browser**:
    Click the link shown in the terminal (usually `http://localhost:5173`) to view the site.

## Build for Production

To create a production-ready build:
```bash
npm run build
```
The output will be in the `dist` folder.
